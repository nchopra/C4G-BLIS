C4G BLIS v2.0Steps for updating to C4G BLIS v2.0 are as follows,		1. Rename folder htdocs in BLIS folder to htdocs_backup	2. Unzip htdocs.zip to BLIS folder	3. Check to see that htdocs folder now contains a list of folders with the first file named C4G BLIS v2.0	4. Now start BLIS and wait till the BLIS login screen pops up	5. Open the file, BLISv2DatabaseUpdate.bat by double clicking on it	6. Now close your browser and then restart BLIS.exe 	Done !
